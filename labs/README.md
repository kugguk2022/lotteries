# Labs

Experiments live here (and adjacent files) and are explicitly **experimental**. Keep runs isolated from the stable API.

## grok.py (root)

- Goal: transformer-on-sequences sandbox used for quick signal-detection ideas.
- Inputs: CSVs under `data/` (`g1.csv`, `poi.csv`) or autogenerated synthetic data if missing.
- Entry: `python grok.py` (no args) â€“ trains a tiny transformer and prints losses. Adjust file paths at the top of the script to point to your data.
- Outputs: training loss logs to stdout; model objects stay in-memory only.

## euromillions_agent/

- Goal: agent + discriminator + grok + CEM mixer for EuroMillions.
- Entry:
  - `python euromillions_agent/lotto_lab.py --csv data/euromillions.csv --mode all --outdir lotto_lab_out`
  - `python euromillions_agent/fetch_prizes.py --out prizes.json` (pull prize tables)
  - `python euromillions_agent/fetch_prizes_range.py --start 2020 --end 2024 --out prizes.json`
- Inputs: normalized EuroMillions CSV from `euromillions.get_draws`, optional prize JSON.
- Outputs: plots + metrics under `--outdir` (calibration, Brier, permutation null histograms).
- Repro: keep the forward-only split, run `lotto_lab.py --mode all --debug` to log intermediate plots.

## euromillions/roi.py

- Goal: EV gating + walk-forward bankroll simulations.
- Status: lab-only; CLI is not exposed yet. Run as a module (`python -m euromillions.roi`) after editing parameters inside the file.
